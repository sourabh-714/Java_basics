Data Types in Java
- Primitive
- Non-Primitive (Reference data type)

- primitive have fixed memory size
- non-primitive follows dynamic memory allocation

- primitive data types do not have references
- non-primitive referes to something

- primitive do not have any methods
- non-primitive have methods

1. Keywords for primitive, no keywords for non-primitive
2. non-primitive derived from primitive    ???????
3. primitive are pre-defined, non-primitve are user defined
4. primitive should have a value, non-primitive can be null
